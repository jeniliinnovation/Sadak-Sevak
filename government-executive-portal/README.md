# City Government Executive Portal

A comprehensive administrative dashboard designed for city government heads and officials to manage public complaints, city operations, infrastructure projects, and staff management.

## Features

- **Executive Dashboard**: Real-time city metrics and KPIs
- **Complaint Management**: Track and manage public complaints
- **Live City Tracking**: Monitor city-wide activities in real-time
- **Staff Management**: Manage government staff and departments
- **Infrastructure Projects**: Track contractors and work orders
- **Performance Analytics**: AI-powered insights and reporting
- **System Audit Logs**: Comprehensive audit trails
- **Role-Based Access**: Secure role and permission management

## Tech Stack

- **React 19** with Vite
- **React Router** for navigation
- **Chart.js** for analytics
- **Leaflet** for mapping
- **Lucide React** for icons

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Building

```bash
npm run build
```

### Preview

```bash
npm run preview
```
